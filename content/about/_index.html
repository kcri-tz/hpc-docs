About the KCRI High-Performance Cluster: History, Hardware, Acknowledgements

{{< /blocks/cover >}} {{% blocks/lead %}} The KCRI High-Performance Cluster isn't big but can do everything we need. {{% /blocks/lead %}} {{< blocks/section >}}

History

The KCRI HPC started out as three Dell Precision 7400 laptop workstations (genie, crick and watson), each sporting a dual quad-core i7-3740QM CPU, 32GB memory and a 1TB HDD.

{{< /blocks/section >}} {{< blocks/section >}}

Features

The HPC now has two entry nodes (genie and crick), two compute nodes (turing, hopper) and a storage node (babbage). A node focused on public access and data sharing (shannon) will be added in 2022.
